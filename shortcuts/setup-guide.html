TLC COLLECTIONS

Siri Shortcuts Setup

Build 4 voice commands for hands-free collection updates

You'll need these for every shortcut:

URL: https://tlc-collections-api.lesthill.workers.dev

API Key: tlc-collect-2026

Open the Shortcuts app on your iPhone to get started. It's pre-installed — search for it if you can't find it.

1 Collections Summary

"Hey Siri, Collections Summary" — hear your total balance, collections, and contacts
  1. Open Shortcuts → tap + (top right corner)
  2. Tap Add Action
  3. Search for Get Contents of URL → tap it
  4. Tap the blue URL field and type:
    https://tlc-collections-api.lesthill.workers.dev/api/summary
  5. Tap Show More (the little arrow under the URL)
  6. Leave Method as GET
  7. Tap Headers → Add new header
    Key: X-API-Key
    Text: tlc-collect-2026
  8. Tap the search bar at the bottom → search for Get Dictionary Value → tap it
  9. Tap Key and type: spoken
  10. Search for Speak Text → tap it
  11. Tap the shortcut name at the very top and rename it to Collections Summary
  12. Tap Done
"Hey Siri, Collections Summary"

2 Collections Update

"Hey Siri, Collections Update" — record a payment by client name
  1. Open Shortcuts → tap +
  2. Add Action → search Ask for Input → tap it
    Prompt: Which client?
  3. Search for Set Variable → tap it
    Variable Name: clientName
  4. Search for Ask for Input again → tap it
    Prompt: How much was collected?
    Tap Input Type → change to Number
  5. Search for Set Variable → tap it
    Variable Name: amount
  6. Search for Get Contents of URL → tap it
    For the URL: type https://tlc-collections-api.lesthill.workers.dev/api/clients/ then tap the Variables button (wand icon above keyboard) → select clientName
  7. Tap Show More
    Change Method to PUT
  8. Tap Headers → Add new header
    Key: X-API-Key   Text: tlc-collect-2026
  9. Tap Body → change to JSON
  10. Tap Add new field → choose Text
    Key: since
    Value: tap the field, then tap Variables → select amount
  11. Search for Show Result → tap it → type: Updated!
  12. Rename to Collections Update → Done
"Hey Siri, Collections Update"

Tip: You can say just a last name — "Gonzalez", "Lucero", "Bishop" — it'll find the right client.

3 Collections Status

"Hey Siri, Collections Status" — mark a client as contacted, committed, collected, or unreachable
  1. Shortcuts → + → Add Action
  2. Ask for Input → Prompt: Which client?
  3. Set Variable → clientName
  4. Search for Choose from Menu → tap it
    Prompt: Set status to:
  5. You'll see "One" and "Two" as placeholders. Tap each to rename them:
    First: Contacted
    Second: Committed
    Tap + Add new item twice more:
    Third: Collected
    Fourth: Unreachable
  6. Under the Contacted section that appeared:
    Add Text action → type contacted
    Add Set Variable → status
  7. Under Committed:
    Text → committed
    Set Variable → status
  8. Under Collected:
    Text → collected
    Set Variable → status
  9. Under Unreachable:
    Text → unreachable
    Set Variable → status
  10. After the End Menu line, add:
    Get Contents of URL
    URL: https://tlc-collections-api.lesthill.workers.dev/api/clients/ + Variable: clientName
    Method: PUT
    Header: X-API-Key = tlc-collect-2026
    Body → JSON → Add field → Text:
    Key: cs   Value: Variable → status
  11. Show Result → Status updated!
  12. Rename to Collections Status → Done
"Hey Siri, Collections Status"

4 Check Client

"Hey Siri, Check Client" — look up any client's balance and status by name
  1. Shortcuts → + → Add Action
  2. Ask for Input → Prompt: Which client?
  3. Set Variable → clientName
  4. Get Contents of URL
    URL: https://tlc-collections-api.lesthill.workers.dev/api/clients/ + Variable: clientName
    Method: GET
    Header: X-API-Key = tlc-collect-2026
  5. Set Variable → client
  6. Get Dictionary Value → Key: nm
  7. Set Variable → name
  8. Get Variable → select client
  9. Get Dictionary Value → Key: bal
  10. Set Variable → balance
  11. Get Variable → select client
  12. Get Dictionary Value → Key: cs
  13. Set Variable → clientStatus
  14. Get Variable → select client
  15. Get Dictionary Value → Key: since
  16. Set Variable → collected
  17. Add Text action and build this sentence using the Variables button to insert each variable:
    [name] has $[balance] remaining. $[collected] collected since workshop. Status: [clientStatus]
  18. Speak Text
  19. Rename to Check Client → Done
"Hey Siri, Check Client"

Sharing With the Team

Once you've built a shortcut, you can share it so others don't have to build it themselves:

  1. Open the shortcut → tap the ··· button
  2. Tap the Share button (bottom of screen)
  3. Tap Copy iCloud Link
  4. Wait for it to generate, then send the link

Anyone who receives the link just taps it to install — no building required.

Client name tips: You can say a first name, last name, or full name. "Gonzalez", "Becky", or "Becky Gonzalez" all work. The system finds the match automatically.

Use anywhere: These work with "Hey Siri" — in the car, in the shower, on a walk. No need to open any app.
